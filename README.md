# 📰 NewsBot — Telegram дайджест

Бот собирает топ постов из публичных Telegram каналов и ранжирует по просмотрам.

## 🚀 Быстрый старт (Windows)

### 1. Установи Python
Скачай с https://python.org (версия 3.10+)
При установке поставь галочку "Add Python to PATH"

### 2. Получи токен бота
- Напиши @BotFather в Telegram
- Отправь /newbot
- Скопируй токен вида: 123456:ABC-xyz...

### 3. Узнай свой Telegram ID
- Напиши @userinfobot в Telegram
- Он пришлёт твой ID (число)

### 4. Заполни .env файл
Открой файл .env и замени значения:
```
BOT_TOKEN=твой_токен_от_BotFather
YOUR_USER_ID=твой_id_от_userinfobot
```

### 5. Запусти бота
Дважды кликни на `start.bat`

---

## 📋 Команды бота

| Команда | Описание |
|---|---|
| /add @channel | Добавить канал |
| /remove @channel | Убрать канал |
| /list | Список каналов |
| /top | Топ прямо сейчас |
| /schedule 09:00 | Авторассылка каждый день |
| /schedule off | Отключить расписание |
| /settings | Настройки |
| /settings 24 10 | Период 24ч, топ 10 постов |

---

## ⚠️ Важно
- Работает только с **публичными** каналами
- Бот должен быть запущен на компьютере для авторассылки
- Время расписания — московское (МСК)
